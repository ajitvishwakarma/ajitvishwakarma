### 👨‍💻 About Me

I'm a **Senior Platform & Java Engineer** with 8+ years of experience building and modernising enterprise-grade distributed systems across **healthcare, fintech, payments, and blockchain** domains.

My most recent work at **Publicis Sapient** sits squarely in the Platform Engineering space  -  I focus on building the *factory*, not just the product:

- 🏗️ **Leading platform modernization of 20+ microservices**  -  a live, multi-sprint initiative applying 16 engineering techniques: AL2023, JDK 21 (Corretto), Tomcat 11 / Jakarta EE, Oracle JDBC 23, Graviton (ARM64), ACM SSL, modular CloudFormation IaC, HashiCorp Vault AppId paths, and AI-assisted migration workflows

- 🐛 **Triaged and resolved 52 JIRA defects** across stage and production environments, including critical issues like Oracle JDBC Kerberos buffer overflow, Vault 403 / credential exposure, and ALB idle timeout regressions

- ⚡ **Reduced service migration effort by 70-80%** through shared tooling, automation patches, and CI/CD standardisation

- 🤖 **AI-assisted development** (GitHub Copilot, VS Code Agents, MCP) achieving **80–90% automation** on repetitive code changes

- 📡 **Architected Kafka Streams pipelines** using a Transformer-Processor pattern for real-time drug communication restriction processing across pharmacy benefit management and claims

> *"I build the paved road so product teams can ship with velocity, safety, and consistency."*
